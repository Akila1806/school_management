# School Management System

## Overview
This project is a school management system designed to help manage student records, classes, schedules, and more.

## Features
- Student Registration
- Class Scheduling
- Attendance Tracking
- Grade Management

## Getting Started
To get a local copy up and running follow these simple steps:

1. Clone the repo
   ```bash
   git clone https://github.com/Akila1806/school_management.git
   ```
2. Install dependencies
   ```bash
   npm install
   ```
3. Run the application
   ```bash
   npm start
   ```
